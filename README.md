# Agentic Honey-Pot System

An AI-powered system that detects scam messages and engages scammers to extract intelligence.

## Features
- Real-time scam detection using pattern matching and AI
- Autonomous AI agent that maintains persona during conversations
- Multi-turn conversation handling
- Intelligence extraction (bank accounts, UPI IDs, phishing links, phone numbers)
- REST API endpoint for integration

## Tech Stack
- **Backend**: FastAPI (Python 3.11+)
- **AI/ML**: OpenAI GPT models / Local LLM options
- **Database**: SQLite (development), PostgreSQL (production)
- **Cache**: Redis (optional)
- **Deployment**: Docker, Uvicorn
- **Authentication**: API Key based
